# ユーザーUI画面の使い方

## 🌐 アクセス方法

ブラウザで以下のURLにアクセスしてください：

```
http://localhost:8000/
```

または直接：

```
http://localhost:8000/static/index.html
```

## 📱 機能説明

### 1. サービス一覧タブ
- 提供されているすべてのサービスを表示
- サービス名、価格、所要時間、説明を確認できます

### 2. 予約するタブ
- 新規予約を作成できます
- 必要な情報：
  - メールアドレス（必須）
  - お名前
  - 電話番号
  - サービス（必須）
  - スタイリスト（オプション）
  - 希望日（必須）
  - 希望時間（必須）
  - 備考

### 3. 予約履歴タブ
- メールアドレスで予約履歴を検索
- 過去の予約情報とステータスを確認できます

### 4. スタイリストタブ
- スタイリスト一覧を表示
- 各スタイリストの専門分野を確認できます

## 🎯 使い方

1. **サービスを確認**
   - 「サービス一覧」タブで利用可能なサービスを確認

2. **予約を作成**
   - 「予約する」タブを開く
   - メールアドレスを入力（既存の顧客の場合は自動で検索されます）
   - サービスを選択
   - 希望日を選択すると、利用可能な時間枠が表示されます
   - 希望時間を選択
   - 「予約を確定」ボタンをクリック

3. **予約履歴を確認**
   - 「予約履歴」タブを開く
   - メールアドレスを入力して「検索」ボタンをクリック

## ⚠️ 注意事項

- サーバーが起動している必要があります
- APIサーバーは `http://localhost:8000` で動作している必要があります
- CORSエラーが発生する場合は、サーバーのCORS設定を確認してください

## 🔧 トラブルシューティング

### ページが表示されない場合
1. サーバーが起動しているか確認
2. `http://localhost:8000/health` にアクセスしてサーバーの状態を確認
3. ブラウザのコンソールでエラーを確認

### データが表示されない場合
1. APIサーバーが正常に動作しているか確認
2. ブラウザの開発者ツール（F12）でネットワークタブを確認
3. APIエンドポイントが正しく応答しているか確認
