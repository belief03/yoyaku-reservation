# 👋 はじめに読んでください

## 📖 このシステムについて

**Yoyaku予約システム**は、個人美容院経営者向けの予約管理システムです。

ホットペッパービューティーや公式LINEと並行して使える、**第三の予約手段**としてご利用いただけます。

## ⚡ クイックスタート

### 1. システムを起動

```bash
python -m api.main
```

### 2. ブラウザでアクセス

- **お客様用画面**: http://localhost:8000/
- **管理画面**: http://localhost:8000/admin
- **APIドキュメント**: http://localhost:8000/docs

---

## 💡 特徴

- ✅ **シンプル**: 1つのコマンドで起動
- ✅ **無料**: 手数料なし
- ✅ **カスタマイズ可能**: 自由に変更可能
- ✅ **24時間対応**: 営業時間外でも予約受付
